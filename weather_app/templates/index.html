That's my Django Weather API App deployed with Railway.

Forecasts

  • You can see forecasts by adding "/forecasts" to the current URL.
  • To show a specific one, you can filter by typing, for example:
    • "/forecasts/?location=Florence"
    • "/forecasts/?location=Florence&date=2023-06-11"
    • "/forecasts/?location=Florence&date=2023-06-11&time=10:30:00"

POST a New Record

To POST a new record, type something like this in the "/forecast" page form:

    { "location": "Florence",
    "date": "2023-06-11",
    "time": "10:30:00",
    "temperature": "25.70",
    "description": "Sunny" }
